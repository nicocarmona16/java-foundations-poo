# java-foundations-poo

## Taller Práctico: Sistema de Gestión de un Refugio de Animales

El objetivo de este taller es desarrollar un sistema para aplicar los pilares de la Programación Orientada a Objetos (POO) y principios de diseño de software.

---

### Objetivos del Taller

1.  Crea una Interfaz llamada Adoptable con un método prepararParaAdopcion().
2.  Crea una Clase Abstracta Animal con atributos encapsulados (nombre, edad, estadoSalud) y métodos getters/setters (Aplica Encapsulamiento).
3.  Crea clases Perro y Gato que hereden de Animal (Aplica Herencia).
4.  Haz que Perro y Gato implementen la interfaz Adoptable y sobrescriban un método hacerSonido() de formas distintas (Aplica Polimorfismo y Abstracción).
5.  Crea una clase Refugio que use colecciones (ArrayList<Animal>) para agregar animales, filtrarlos por edad y simular una adopción.
6.  Manejo de Excepciones: Lanza un error personalizado si alguien intenta adoptar un animal que no está sano.
